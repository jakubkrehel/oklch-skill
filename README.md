# OKLCH Skill

> [!IMPORTANT]
> **This skill has moved to [jakubkrehel/skills](https://github.com/jakubkrehel/skills)**, together with my other agent skills. This repo is kept for existing installs but no longer receives updates.

Install the current version from the new repo:

```bash
npx skills add jakubkrehel/skills --skill oklch-skill
```

---

An [Agent Skill](https://docs.anthropic.com/en/docs/claude-code/skills) for working with the OKLCH color space in web projects. Based on [oklch.fyi](https://oklch.fyi). It teaches AI coding assistants (Claude Code, Codex, etc.) to use OKLCH — the perceptually uniform color space that eliminates hue drift, makes palette generation predictable, and simplifies accessibility fixes.
